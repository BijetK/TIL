# 1. branch 생성

```bash
$ git branch 브랜치 이름
```

# 2. branch 이동

```bash
$ git checkout 브랜치 이름
```

# 3. branch 생성 및 이동

```bash
$ git checkout -b 브랜치 이름
```

# 4. branch 조회

```bash
$ git branch
```

# 5. branch merge

```bash
(master) $ git merge 브랜치 이름
```

- master로 병합하여는 경우 master 브랜치에서 명령어를 작성해야함!!
- `브랜치 이름`을 해당 브랜치로 병합하는 명령어이기 때문

# 6. branch 삭제

```bash
$ git branch -d 브랜치 이름
```
